Ask 485 Search Engine

Team size: 3
When: November-Decemeber 2023
About: We created a 7 stage mapreduce pipeline using Python to rank documents according to text and link analysis. We coded a REST API using Flask that made requests to the index server when the user inputted a query and page rank (shown via slider). This request allows our Ask485 large scale search engine to display the top 10 most relevant websites for the user. We also used HTML, CSS, and JavaScript to create the user interface. Lastly, we incorporated shell scripts and threading to manage three servers in a simulated distributed system on local host.
Demo: YouTube video and code gists
