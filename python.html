March Madness Bracket Generator

Team size: 5
When: February-April 2024
About: Our team created a bracket generator for March Madness for our semester long project. We coded a web crawler to scrape NCAA data from the ESPN website. Then we created a custom score metric using win percentage and SOS (Strength of schedule). Then I personally designed algorithms in Python to predict the winners of each game in every round using seeding history and our metric. I incorporated randomness so we could generate a variety of brackets which averaged a precision of 34% for the final four over 50 brackets (which is better than the average human). We quantified and visualized our results in the following poster as well as a written report.

Mapreduce Distributed System

Team size: 3
When October-November 2023:
About: We simulated a distributed systems environment using TCP and UDP threading and sockets. We executed the Michigan Hadoop algorithm called Madoop. In order to properly simulate a distributed environment we had to create a layer of abstraction. Our Manager class had to assign jobs to remote Workers and reassign tasks if the Worker was offline. Each Worker had to complete its given task and send heartbeats using UDP to the Manager. There was no direct connection between the Manager and workers.
