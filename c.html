LC2K Pipeline Simulator

Team size: 1
When: November 2022-December 2022
About: I used C to simulate a computer reading tasks in binary. This program can handle the following instructions: add, nor, lw (load word), sw (save word), beq (conditional branch), jalr (jump), and halt. This machine also has a cache, processor, and memory simulated with C data structs and methods. I utilized data forwarding and branch prediction to resolve hazards and speed up the program. I designed my own test cases in machine code for this project.

Treasure Hunt Game

Team size: 1
When: Janurary 2022
About: I built a text based game in C++ where pirates are searching for treasure on varied terrain. My program uses conditionals to identify land, water, obstacles and treasure terrain. If the ship is on water the Captain searches and if on land the First Mate searches. The Captain and First Mate are assigned a breadth first or depth first search strategy based on user input. Once the treasure is found the program outputs a direct path from the starting point to the treasure. This project utilizes object-oriented programming (OOP) structures stack and queue. I was given a spec for the project but had full freedom in how to create classes and approach the problem. I designed my own test cases and treasure maps for this project
